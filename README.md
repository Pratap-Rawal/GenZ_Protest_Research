# Media Framing of Gen Z Protests in Nepal

This project analyzes how media framing evolved during the 2025 Gen Z protests in Nepal using content analysis of news articles.

## 🎯 Objective

To examine how media narratives changed as events escalated, particularly in terms of:
- Tone
- Framing (Law & Order vs State Repression)
- Focus (Protestors vs Government)
- Attribution of violence

## 📊 Dataset

The dataset consists of coded news articles from The Kathmandu Post.

Each article is analyzed based on:
- Tone
- Frame
- Focus
- Violence emphasis

## 🧠 Key Findings

- Initial coverage focused on **law and order**
- After fatalities, framing shifted to **state repression**
- Media focus moved from **protestor actions → state accountability**
- Later reports included **forensic evidence and accountability narratives**
